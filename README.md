# 🎨 AI Workshop — Track Frontend

Задание: интерактивный дашборд на чистом HTML/CSS/JS без фреймворков.

Требования:
- Карточки: выручка, заказы, средний чек, топ-продукт
- График по месяцам на Canvas/SVG
- Таблица с сортировкой
- Фильтры по продукту и периоду
- Тёмная тема
- Адаптивность

## Работа с данными (`data/sales.json`)

По умолчанию проект должен работать локально без сервера. Поэтому рекомендуемый вариант:

- встроить данные прямо в JS-файл, например:
  `const SALES_DATA = { ... }`

Альтернатива (если хотите использовать `fetch`):

```bash
python -m http.server 3000
```

После этого открыть `http://localhost:3000`.

Оба варианта допустимы, но в README участника должен быть явно указан выбранный способ.

Бонус: PWA с offline-режимом.
